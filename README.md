# 趣友 appUI 二开

蓝湖：[趣友](https://www.quyouapp.com/)

## 项目介绍

> 趣友 appUI 二开，基于 uniapp+vue3+ts 开发，支持多端编译，包括微信小程序、H5、APP 等。
> 环境：node:v22.17.0; pnpm：10.13.0;
> Vue3+TS+Uniapp;

## 项目启动 测试

```bash
# 安装依赖
pnpm install
pnpm run dev
```

## 项目结构

> 登录；信息填写；
> 主页:个人信息；发送消息；
> 动态：个人信息；动态内容；评论内容；
> 消息：消息列表；消息详情；
> 我的：个人信息；设置；
